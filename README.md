# Hobby Devs
## Vision
Hobby Devs is about creating a collaborative space, where ideas meet development. We want to connect people with innovative ideas to hobby developers who can bring these visions to life. Users can post their project ideas and developers can apply, providing their budget and time frame. We want to create a space, where developers can ask each other for advice and help. Through ratings, the project idea can find the perfect developer, that will help you build your idea to reality.

We want to create an easy application process, where project owners can submit forms detailing their needs and developers can apply to these projects. Developers will be able to create profiles, where they can state their experiences and skills, helping project owners understand what they offer. Through ratings, the process of choosing the perfect developer for projects is efficient and easy.
